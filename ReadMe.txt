# GIT Basics


### Initialize git
`git init`   - creates a local repository in that directory.


### get the status of your repository
`git status`

## How to commit

### First make some changes like adding/updating/deleting a file

### Add that that file into staging
`git add <filename>`

### commit all the changes
`git commit`

or 

`git commit -m 'Commit message'`

### get git version 
`git --version`
